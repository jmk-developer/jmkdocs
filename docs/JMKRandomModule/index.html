JMK Random Module

note

You can grab this module here, or require it using the ID 5735904266.

The JMK Random Module is a generator for the JMK Random object. This module can be used as a direct replacement for the default Random global.

local Random = require(script.Parent.MainModule)

or with set and getfenv.

local env = getfenv(0)
env["Random"] = require(script.Parent.MainModule)
setfenv(0, env)
env = nil
  • Random.new(Seed)
    This generates a new instance of JMK Random. If no seed is supplied, a seed from an internal entropy source will be used.
