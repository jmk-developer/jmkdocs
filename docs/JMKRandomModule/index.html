JMK Random (generator)

note

You can grab this module here, or require it using the ID 5735904266.

The JMK Random Module is a generator for the JMK Random object. This module can be used as a direct replacement for the default Random global.

important

The code shown below will only work in a Script, due to the fact that LocalScripts do not support requiring modules by ID. To use this module in a LocalScript, clone the module and place it in ReplicatedStorage and require that instead.

It can be used by redefining Random as a local variable, like this:

local Random = require(5735904266)

or editing the script's environment variables, like this:

local env = getfenv(0)
env["Random"] = require(5735904266)
setfenv(0, env)
env = nil
  • Random.new(Seed)
    This generates a new instance of JMK Random. If no seed is supplied, a seed from an internal entropy source will be used.
