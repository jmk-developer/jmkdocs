JMK Face Module

caution

This is a private module and is currently not public. This documentation exists for those who can access the module.

The JMK Face Module is a class-based module that allows manipulation and modification of a character's face.

Generators#

  • JMKFaceModule.generateBases()
    Returns a new Mouth and Eyes object that can be used for other functions.
  • JMKFaceModule.new(MouthObject, EyesObject)
    Returns a newly generated JMKFace instance using the supplied objects.
  • JMKFaceModule.fromHead(Head)
    Returns a newly generated JMKFace instance using the supplied character's head. This will destroy the old face inside the head.
  • JMKFaceModule.fromCharacter(Character, IgnoreHumanoid)
    Returns a newly generated JMKFace instance using the supplied character. This wil destroy the old face inside of the character's head just like JMKFaceModule:fromHead.
  • JMKFaceModule.fromPlayer(Player)
    Returns a newly generated JMKFace instance using the supplied player. This has the same functionality as JMKFaceModule:fromCharacter and will also destroy the previous face.
