JMK Random

warning

This is not a standalone module. To generate this module, use the JMK Random Module.

The JMK Random Module is a simple, easy to use module that can be used in place of the Random object.
It contains shorthand for many different Roblox Lua types, such as UDim, Vector3, and many others. You can use it in place of the Random module by redefining the Random object

local Random = require(script.Parent.MainModule)

or with set and getfenv.

local env = getfenv(0)
env["Random"] = require(script.Parent.MainModule)
setfenv(0, env)
env = nil
  • Random.new(Seed)
    This generates a new instance of JMK Random. If no seed is supplied, a seed from an internal entropy source will be used.
