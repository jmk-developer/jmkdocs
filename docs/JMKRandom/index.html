JMK Random

warning

This is not a standalone module. To generate this module, use the JMK Random Module.

JMK Random is a simple, easy to use module that can be used in place of the Random object, generated by the JMK Random Module with the Random.new function.
It contains shorthand for many different Roblox Lua types, such as UDim, Vector3, and many others.

Numbers#

  • JMKRandom:NextInteger(Minimum, Maximum)
    Returns a random integer ranging from Minimum to Maximum.
  • JMKRandom:NextNumber(Minimum, Maximum)
    Returns a random number ranging from Minimum to Maximum.

NumberRange#

  • JMKRandom:NextIntegerFromNumberRange(NumberRange)
    Returns a random integer ranging from the supplied NumberRange minimum and maximum value.
  • JMKRandom:NextNumberFromNumberRange(NumberRange)
    Returns a random number ranging from the supplied NumberRange minimum and maximum value.

Vectors#

Vector3#

  • JMKRandom:NextVector3(Vector3Min, Vector3Max)
    Returns a random Vector3 ranging from Vector3Min and Vector3Max. The resulting Vector3 will be made up of numbers, not integers.
  • JMKRandom:NextIntVector3(Vector3Min, Vector3Max)
    Same as JMKRandom:NextVector3 but all values are integers.
  • JMKRandom:NextEvenVector3(Minimum, Maximum)
    Same as JMKRandom:NextNumber but it returns a Vector3 with identical values.

Vector2#

  • JMKRandom:NextVector2(Vector2Min, Vector2Max)
    Returns a random Vector2 ranging from Vector2Min and Vector2Max. The resulting Vector2 will be made up of numbers, not integers.
  • JMKRandom:NextIntVector2(Vector3Min, Vector3Max)
    Same as JMKRandom:NextVector2 but all values are integers.
  • JMKRandom:NextEvenVector2(Minimum, Maximum)
    Same as JMKRandom:NextNumber but it returns a Vector2 with identical values.

Colors#

  • JMKRandom:NextColor3(Color3Min, Color3Max)
    Returns a random Color3 ranging from Color3Min and Color3Max.

UDims#

  • JMKRandom:NextUDim(UDimMin, UDimMax)
    Returns a random UDim ranging from UDimMin and UDimMax. Numbers are used for the Scale property and integers are used for the Offset property.
  • JMKRandom:NextUDim2(UDim2Min, UDim2Max)
    Same as JMKRandom:NextUDim but it returns a UDim2.
